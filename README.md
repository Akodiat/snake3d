# Running the game locally
Start a local static webserver in this directory. If you have python 3, you can type:
`python -m http.server 8080`

A full list of static server oneliners is available here:
https://github.com/imgarylai/awesome-webservers

You will then be able to run the game at http://localhost:8080/
